# 🏪 店舗DX - 簡単使い方ガイド

## 🚀 クイックスタート(3ステップ)

### 1️⃣ GitHubにアップロード
```
index.html をGitHubリポジトリのルートに配置
```

### 2️⃣ GitHub Pagesを有効化
```
Settings > Pages > Source: "Deploy from a branch" > Branch: main > Save
```

### 3️⃣ ブラウザで開く
```
公開されたURLにアクセス(例: https://username.github.io/repository-name/)
```

---

## 📝 実際の使用手順

### ステップ1: 店舗マスタを設定
画面の「店舗マスタ設定」エリアに、以下の形式で店舗情報を入力:

```
店舗コード,店舗名,出荷曜日
0089,西宮,金
0171,イオン伊丹,金
0026,ＬＴ,水
```

> 💡 **ポイント**: デフォルトで16店舗分が既に設定済みです

---

### ステップ2: Excelファイルを選択
以下のいずれかの方法でファイルを選択:
- **ドラッグ&ドロップ**: ファイルを画面の青い枠内にドロップ
- **クリック選択**: 青い枠をクリックしてファイル選択ダイアログから選択

---

### ステップ3: 自動変換&ダウンロード
✅ ファイルが自動的に変換されます
✅ ZIPファイルが自動的にダウンロードされます

## 💡 よくある質問

### Q1: 店舗マスタは毎回設定が必要ですか?
**A**: いいえ。ブラウザに保存されるため、次回以降は設定不要です。

### Q2: 対応しているExcelのバージョンは?
**A**: .xlsx形式と.xls形式の両方に対応しています。

### Q3: ファイルの最大サイズは?
**A**: ブラウザの制限内であれば問題ありません(通常10MB程度まで)。

### Q4: オフラインでも使用できますか?
**A**: はい、一度ページを読み込めばオフラインでも動作します。

### Q5: スマートフォンでも使えますか?
**A**: はい、モバイルブラウザでも使用可能ですが、PC推奨です。

## 🔧 トラブルシューティング

| 問題 | 解決方法 |
|------|---------|
| ZIPがダウンロードされない | ブラウザのポップアップブロックを解除 |
| エラー: "店舗マスタを設定してください" | 店舗マスタのテキストエリアに情報を入力 |
| 変換B-2にデータが出力されない | 該当店舗コードがマスタに登録されているか確認 |
| 文字化けする | UTF-8対応のエディタで開く(Excel推奨) |
